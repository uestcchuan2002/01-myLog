## 6月11日日志

### 1. 力扣刷题

今天的力扣刷题主要解决了两类问题分别是，排列问题和棋盘问题：
对应排列问题为：
- 46.全排列1
- 47.全排列2

> 排列问题的话，就是给定的数组中的每个元素都要使用的到，因此就不需要参数`index`了，直接每次都从`i = 0`开始，但是相对于全排列1，全排列2需要我们进行去重，这里的思路就是和之前的无异：
```c
if (i > 0 && nums[i] == nums[i - 1] && used[i - 1] = false) {
    continue;
}
```

对应的棋盘问题为：
- 51.N皇后
- 37.解数独

> 我觉得棋盘问题的难点主要体现在以下两个方面
> - 不容易发现是通过回溯算法解决
> - 它的判断依据比较难，比如N皇后题目是判断同行，同列，同对角线没有相同元素，这其实需要较器的基本功，只有足够熟练才能快速写出

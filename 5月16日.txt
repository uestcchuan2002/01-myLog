1.力扣刷题
	今天刷了5到有关链表的题目，对链表的更加熟悉了。其中最有印象深刻的一道题目是142题，环形链表2。
	这个题目的题意是，判断一个链表中是否存在环形，如果存在的话，输出环形的入口点，如果不存在的话，输出null;
	这个题的做法非常的有创意：
	1.先使用快慢指针，判断是否有环形。
	2.如果有环形的话，快慢指针重合，此时进行入口点的查询。
	3.当快慢指针重合时，slow(慢指针)，fast(快指针)
	slow = a + b;
	fast = a + b + nx;
	其中 a：起始点到入口点的距离
	b: 入口点到重合点的距离
	n: fast套的slow的圈数
	x: 一个环的长度；
	因为：fast的速度是slow的两倍，因此
	a + b + a + b = a + b + nx;
	结论： nx  = a + b;
	4.此时从起始点准备一个slow2，与slow1同时开始走
	5.当slow2走到入口点时,走了a,slow 走了 a + b + a;
	由结论 a + b = nx可知，此时slow 与 slow1重合，
	由此可知，当slow与slow2重合的时候，就是入口点
